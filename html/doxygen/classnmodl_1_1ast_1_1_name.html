Detailed Description

Represents a name.

Whenever lexer encounters string variable, it returns a ast::Name type. So, along with ast::Integer, ast::Double ast::String and ast::PrimeName, ast::Name is one of the fundamental type in the AST. Many other variable types (e.g. ast::GlobalVar, ast::RangeVar) have underlying value stored as ast::Name.

Note
This node should be able to use std::string as value type instead of ast::String
