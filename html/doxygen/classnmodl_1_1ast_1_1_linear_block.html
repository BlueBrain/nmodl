Detailed Description

Represents LINEAR block in the NMODL.

A set of simultaneous equations can be specified in a LINEAR block. Here is an example :

LINEAR clamp {
LOCAL t1, t2
t1 = tau1/dt
t2 = tau2/dt
~ vi = v + fac*vo - fac*v
~ t2*vo - t2*vo0 + vo = -gain * e
~ -stim - e + vi - e + t1*vi - t1*e - t1*(vi0 - e0) = 0
}
