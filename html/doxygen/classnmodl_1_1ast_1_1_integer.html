nmodl::ast::Integer Class Reference
AST Implementation » AST Classes

Represents an integer variable. More...

Detailed Description

Represents an integer variable.

Non floating value in the mod file is parsed as an integer. For example, in the below code, integer literals like 0 and 11 are stored as ast::Integer in the AST :

FROM i=0 TO N {
tau = X[i] + 11
}
See also
